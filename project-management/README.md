# SmartKhata Project Management Baseline

Baseline date: 2026-06-23

This folder stores the working roadmap, trackers, and reports for SmartKhata.
All `.tsv` files are tab-separated and can be copied directly into Google Sheets.

Update rule:
- Do not recreate the roadmap from scratch.
- Update existing rows when work progresses.
- Add new rows only when new scope appears.
- Move status forward through: To Do, In Progress, Blocked, QA, Done.
- Keep completion percentages realistic and tied to verified work.

Suggested daily flow:
1. Update `02_Daily_Updates.tsv` with completed work, blockers, actual hours, and completion percent.
2. Update `03_Daily_Task_Planner.tsv` for today, tomorrow, and the day after.
3. Update relevant module trackers.
4. Refresh `16_Daily_Report.tsv`, `17_Weekly_Report.tsv`, and `18_Release_Report.tsv`.
